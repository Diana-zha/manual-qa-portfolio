


## BUG-XX: Текст сообщения об ошибке выходит за границы контейнера на странице авторизации

**Environment / Окружение:**
- Браузер:
- Операционная система:
- Viewport / Размер области просмотра:

**Preconditions / Предусловия:**
- Открыта страница авторизации SauceDemo.

**Steps to Reproduce / Шаги воспроизведения:**
1. Открыть страницу авторизации SauceDemo.
2. Установить узкую ширину окна браузера.
3. Ввести невалидные значения в поля `Username` и `Password`.
4. Нажать кнопку `Login`.

**Actual Result / Фактический результат:**
Текст сообщения об ошибке не помещается внутри красного контейнера и выходит за его визуальные границы. Из-за этого сообщение отображается некорректно и ухудшается его читаемость.

**Expected Result / Ожидаемый результат:**
Текст сообщения об ошибке должен полностью помещаться внутри контейнера, корректно переноситься на новые строки и не выходить за его границы.

**Severity / Серьёзность:**
Low

**Priority / Приоритет:**
Medium

**Attachments / Вложения:**
