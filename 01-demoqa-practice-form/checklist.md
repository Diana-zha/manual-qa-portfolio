# Checklist

## Feature

DemoQA Student Registration Form

## Checklist

### Отправка формы

- [ ] Проверить успешную отправку формы при заполнении обязательных полей First Name, Last Name, Gender и Mobile валидными данными.
- [ ] Проверить успешную отправку формы при заполнении всех полей формы валидными данными.
- [ ] Проверить, что после нажатия Submit при валидных данных отображается модальное окно подтверждения.

### Проверка обязательных полей

- [ ] Проверить, что форма не отправляется, если поле First Name пустое.
- [ ] Проверить, что форма не отправляется, если поле Last Name пустое.
- [ ] Проверить, что форма не отправляется, если поле Gender не выбрано.
- [ ] Проверить, что форма не отправляется, если поле Mobile пустое.

### Проверка полей формы

- [ ] Проверить, что если поле Email заполнено, система валидирует его формат.
- [ ] Проверить, что форма не отправляется при вводе Email в невалидном формате.
- [ ] Проверить, что поле Mobile считается валидным, если содержит ровно 10 цифр.
- [ ] Проверить, что форма не отправляется, если поле Mobile содержит менее 10 цифр.
- [ ] Проверить, что форма не отправляется, если поле Mobile содержит более 10 цифр.
- [ ] Проверить, что поле Mobile не принимает буквы и специальные символы.
- [ ] Проверить, что пользователь может заполнить поле Current Address.

### Дата рождения

- [ ] Проверить, что пользователь может выбрать Date of Birth через date picker.
- [ ] Проверить, что выбранная Date of Birth отображается в поле формы.

### Предметы и хобби

- [ ] Проверить, что пользователь может выбрать Subject из списка автодополнения.
- [ ] Проверить, что пользователь может выбрать одно значение Hobbies с помощью чекбокса.
- [ ] Проверить, что пользователь может выбрать несколько значений Hobbies с помощью чекбоксов.

### Загрузка изображения

- [ ] Проверить, что пользователь может выбрать файл в поле Picture.
- [ ] Проверить, что имя выбранного файла отображается в форме после загрузки.
- [ ] Проверить поведение формы при загрузке файла неподдерживаемого формата.

### Штат и город

- [ ] Проверить, что пользователь может выбрать State из выпадающего списка.
- [ ] Проверить, что пользователь может выбрать City из выпадающего списка после выбора State.
- [ ] Проверить, что список City зависит от выбранного State.

### Модальное окно подтверждения

- [ ] Проверить, что после успешной отправки формы отображается модальное окно подтверждения.
- [ ] Проверить, что в модальном окне подтверждения отображаются данные, введённые или выбранные пользователем в форме.
- [ ] Проверить, что модальное окно подтверждения закрывается по кнопке Close.

### Интерфейс и доступность

- [ ] Проверить, что обязательные поля визуально подсвечиваются при попытке отправить пустую форму.
- [ ] Проверить, что по форме можно переходить с помощью клавиши Tab в логичном порядке.
